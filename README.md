# gm_geoip
GModule for acquiring GeoIP data using [libmaxminddb](http://maxmind.github.io/libmaxminddb/). Intended for use with the [GeoLite2 City](http://dev.maxmind.com/geoip/geoip2/geolite2/#Downloads) database.
